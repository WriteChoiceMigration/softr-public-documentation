## Adding your website

To integrate Crisp on your Softr app, you need to add your site domain and name when registering on Crisp or in **Settings => Website Settings => Add a new website**.
<Frame caption="Add a new website">![Add a new website](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fdaeea69c-e4d4-4ec5-8d48-defd64fef7f0%2Fadd-new-website.png&blockId=61edafb8-ea0e-43dc-8f37-214bf836c1c0&projectId=1032)</Frame>

Add your website details as shown below.
<Frame caption="Adding website details">![Adding website details](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F02449c4b-1f1d-43d4-a873-72107c37413e%2Fadding-website-domain.png&blockId=d6a0e36c-e004-431c-b630-6324a6c6f8ff&projectId=1032)</Frame>

## Getting the website ID

As soon as your site is set up, go ahead and connect Crisp. First, you need to click **Integrations**
next to your site name in **Settings => Website settings**.
<Frame caption="Integrations">![Integrations](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F9cce4efa-c6ac-4e43-97e2-d26adeda7229%2Fintegrations.png&blockId=554cffc9-c1af-4b67-b231-b4b2a09e5e24&projectId=1032)</Frame>

In the pop-up that appears next, choose the **HTML** option.
<Frame caption="Selecting the HTML option">![Selecting the HTML option](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Ffeae4995-7041-436e-880b-96bc10a764cb%2Fhtml.png&blockId=557ecc3a-4f12-4ffc-a7bf-215e6d9116af&projectId=1032)</Frame>

In the HTML code, locate the **Website ID** and copy it.
<Frame caption="Locating Website ID">![Locating Website ID](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F807a2d44-0995-492d-a0e0-7bd327fe14b9%2Fwebsite-id.png&blockId=1f80884e-48ee-417e-846e-a9bb0a582b87&projectId=1032)</Frame>

## Adding the Website ID in Softr Studio

Finally, go back to Softr Studio and paste the code in **Settings => Integrations => Crisp**.
<Frame caption="Adding Website ID in Softr">![Adding Website ID in Softr](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fc9c69b32-3c11-437a-a4f6-c969612adcea%2Fadding-id-in-softr.png&blockId=4ee3e9ac-a530-407b-9532-2c9386a28f6d&projectId=1032)</Frame>

Save the settings, publish your app again, and you’re done. Now, the Crisp chat icon should appear on your site, allowing your customers to contact you at any time.
<Frame caption="Crisp Chat activated on the site">![Crisp Chat activated on the site](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F76107299-282b-45cb-b45c-a956f7bf0de0%2Fcrisp-chat-activated.png&blockId=cadf68eb-1a2c-4fcb-a7b4-ede2d88e4e02&projectId=1032)</Frame>
