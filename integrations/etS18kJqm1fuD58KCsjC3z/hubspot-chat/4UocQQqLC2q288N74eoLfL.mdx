## Creating a Chatflow

To add a chat widget to your Softr app, you first need to create a chatflow in your HubSpot account to be able to receive messages. To do that, go to **Conversations => Chatflows => Create chatflow.**
<Frame caption="Creating a Chatflow">![Creating a Chatflow](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F489a5b18-a374-40f6-ba90-74b07d8edd24%2Fcreating-chatflow.png&blockId=ceda6f3a-0d80-41ab-916d-50fa50ca422c&projectId=1032)</Frame>

In the next step, choose **Website**.
<Frame caption="Choosing the “Website” option">![Choosing the “Website” option](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Faef8bb8f-d6b1-4c61-970b-b38c8f8b4af3%2Fchoosing-website-option.png&blockId=1ea0139b-f39e-402e-a762-c207d1c94394&projectId=1032)</Frame>

Then select **Live chat**.
<Frame caption="Selecting the “Live chat”option">![Selecting the “Live chat”option](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fe0e48bd7-62f4-4c70-823a-13a2fc35a00f%2Fselecting-live-chat.png&blockId=c342a09b-762b-44f2-8ecc-baae4276348d&projectId=1032)</Frame>

Next, select the **Inbox** and a **Language** for the chatflow, and hit **Create**. You can find more info on configuring these and other parameters of the chatflow [here](https://knowledge.hubspot.com/chatflows/create-a-live-chat).
<Frame caption="Selecting Inbox and Language">![Selecting Inbox and Language](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fb17ec3d8-4c47-47b0-b21b-3b7665f3b73b%2Fselecting-inbox-and-language.png&blockId=f1a74ac6-2f0c-4451-8c5f-34839a5424ff&projectId=1032)</Frame>

Lastly, you enter the edit page where you can [customize](https://knowledge.hubspot.com/chatflows/create-a-live-chat) your chatflow, and, as soon as you are done, enable it in the upper right corner of the page.
<Frame caption="Enabling chatflow">![Enabling chatflow](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F31746c7e-d287-4ca6-84cf-97df6906ca1d%2Fenabling-chatflow.png&blockId=4db61696-10d3-4918-a399-b9f35371a0ac&projectId=1032)</Frame>

So, your chatflow is ready, and you can go ahead and add the tracking code in your Softr app to start displaying the chat widget to your visitors.

## Getting the Tracking Code

Go to **Settings** in your HubSpot account.
<Frame caption="Accessing settings">![Accessing settings](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F75f6ba6e-695a-4bfe-adb0-e377cf6aa3fa%2Faccessing-settings.png&blockId=932d8b5b-ec7a-4c8f-ad3b-5039fbe7e039&projectId=1032)</Frame>

Scroll down to **Tracking Code** and copy the **Embed code**.
<Tip>The location of the tracking code can be different depending on your HubSpot subscription plan. Check out this [article](https://knowledge.hubspot.com/reports/install-the-hubspot-tracking-code) for more details.</Tip>
<Frame caption="Getting the tracking code">![Getting the tracking code](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F680c288d-82fb-4786-b031-7b83457e7232%2Fgetting-tracking-code.png&blockId=88ff7c2d-531f-4246-9433-3c857eb0a845&projectId=1032)</Frame>

## Adding the tracking code in Softr studio

In Softr Studio, go to **Settings => Integrations => HubSpot Chat**, and enter the **Tracking Code**.
<Frame caption="Adding the tracking code in Softr">![Adding the tracking code in Softr](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F1d368267-5259-456c-8163-44f735eadc83%2Fadding-tracking-code-to-softr.png&blockId=c95bccc7-e018-4aeb-ab31-5e1c97029b7a&projectId=1032)</Frame>

Save the **Settings**, republish your app, and you're done. Now, the chat widget should start appearing on your site.
<Frame caption="HubSpot chat widget added">![HubSpot chat widget added](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F0d57d9f9-c328-44b2-bda6-13dc460e1699%2Fchat-widget-added.png&blockId=36760238-c53b-4893-9e18-44cede0b984f&projectId=1032)</Frame>
