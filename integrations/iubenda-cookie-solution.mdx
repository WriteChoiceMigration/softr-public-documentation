Iubenda Cookie Solution allows to add a customizable cookie consent banner on your site. You will need an Iubenda subscription for this integration. Let's see how it can be done.

<Tip>By default, Iubenda will block the most common scripts on your site until the user hits **Accept**. More details on privacy controls and cookie solution configuration are available [here](https://www.iubenda.com/en/help/1205-how-to-configure-your-cookie-solution-advanced-guide).</Tip>

## Generating a Banner

In your Iubenda account home page, click the **Start Generating** button.
<Frame caption="Start generating">![Start generating](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F3597a5d0-7254-4c9c-b485-3e0d582f5ea7%2Fstart-generating.png&blockId=c97e7710-6d2d-49ac-868e-6474867871de&projectId=1032)</Frame>

Choose the **Website** option, specify a **Site Name**, and hit **Start Generating**.
<Frame caption="Choosing the “Website” option">![Choosing the “Website” option](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F31cbd83c-4853-4b6b-b63b-8a363a5f9ce0%2Fgenerating-code.png&blockId=822e383a-0309-4d5f-8415-9c63941ff9c0&projectId=1032)</Frame>

Next, locate the **Cookie Solution** option and click **Generate Now**.
<Frame caption="Generating the code">![Generating the code](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F46079991-485e-407f-a394-cb28f44cff47%2Fgenerate-now.png&blockId=48803e18-60df-4fe5-bd17-ec3b1fb052a3&projectId=1032)</Frame>

In the following page, you can customize your banner's design and configure other settings. Hit **Next** as soon as you're done.
<Frame caption="Customizing the banner design">![Customizing the banner design](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F818cc7ec-c844-4d2b-873d-318e90a5a73e%2Fcustomizing-banner-design.png&blockId=5791f040-7371-4de2-afcc-e4684cf6e052&projectId=1032)</Frame>

Next, if you don't have a paid license, you'll see a pop-up with the upgrade options. You don't need a paid license for the cookie solution, so, if you don't want to upgrade at this point, simply close the pop-up.
<Frame caption="Closing the upgrade pop-up">![Closing the upgrade pop-up](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F1f46e8c5-1c19-4026-bacf-b7ebdb619c7b%2Fclose-upgrade-pop-up.png&blockId=3c32015b-fd1c-4d43-8361-4f7a4d58996d&projectId=1032)</Frame>

## Getting the Banner Code

Now, you should see an **Embed** button under **Cookie Solution**. Click the button to get the banner code.
<Frame caption="Getting the embed code">![Getting the embed code](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fadba31c8-b613-4f9b-b1c3-e3d0386b39ac%2Fgetting-embed-code.png&blockId=6d439400-e683-49ae-9ba4-1f3b26b2a97a&projectId=1032)</Frame>

Here, you can enable/disable **Prior blocking and asynchronous re-activation** and customize the banner further if needed. When everything is ready, simply click **Copy** to copy the banner code.
<Frame caption="Accessing the embed code and customizing the banner further">![Accessing the embed code and customizing the banner further](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Ff54eaaa5-3772-4f1d-ac22-9cbab7465ee7%2Faccess-embed-code-and-customize.png&blockId=37eb7491-5d78-45aa-a311-afb45b727cfc&projectId=1032)</Frame>

## Adding the Banner Code in Softr Studio

Now, switch to your Softr Studio, navigate to **Settings => Integrations => Iubenda Cookie Solution,** and paste the code.
<Frame caption="Applying banner code in Softr">![Applying banner code in Softr](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fde29919d-4f19-434f-a33a-15a6bcfd9d38%2Fadding-banner-code-softr.png&blockId=5a18c366-effb-4ca5-a997-3916edb318b0&projectId=1032)</Frame>

You're almost done. Just save the settings and republish your app, and you'll notice the cookie consent banner appearing.
<Frame caption="Iubenda pop-up published on the Softr app">![Iubenda pop-up published on the Softr app](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F66311a08-823a-40c9-bb75-7efc90350a2e%2Fbanner-published.png&blockId=e8317d41-f4f3-4dea-b2ba-3f7ce3278d8f&projectId=1032)</Frame>
