## 

Here we’ll go through the process of adding the Drift widget to your site. In your Drift account, you need to go to **Settings => Drift widget**, scroll down to **Install**, and locate the **JavaScript** option.
<Frame caption="Drift widget setup">![Drift widget setup](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F75edf392-9b01-49f3-8892-d9f2a2b09a35%2Fdrift-widget.png&blockId=7f00de7e-f93c-4e29-a97f-dcc698398fc9&projectId=1032)</Frame>

Hit **Copy** to copy the JavaScript code.
<Frame caption="Copying the JS code">![Copying the JS code](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Fae08c68b-6482-4843-a44e-e63373432af1%2Fcopying-js-code.png&blockId=8fca3209-f574-4037-9bfa-b6ca32c9278d&projectId=1032)</Frame>

In your Softr Studio, go to **Settings => Integrations => Drift Chat** and paste the code inside the field.
<Frame caption="Adding Drift widget code in Softr">![Adding Drift widget code in Softr](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2Feb67655c-9b40-49d6-8211-4c37f4fe5fdb%2Fadding-drift-code-in-softr.png&blockId=c913797e-6cbf-4837-9091-7d451757b37e&projectId=1032)</Frame>

That’s it. As soon as you save the updated settings and republish your app, you’ll see the Drift widget appear on the site, as shown below.
<Frame caption="Drift widget published on site">![Drift widget published on site](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F55c492ba-a567-41ee-810e-3f13b190beba%2Fdrift-widget-activated.png&blockId=f0ab9343-8c92-4716-a9ef-1a24d0aae227&projectId=1032)</Frame>
