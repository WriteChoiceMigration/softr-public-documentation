<Note>Inbox block is available for users on the [Professional plan and above](https://www.softr.io/pricing).</Note>

The Inbox layout works similar to an email inbox where you select an item from the sidebar on the left side of the screen and its details are displayed inside the window next to it dynamically.
<Frame>![Notion image](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F9d1a1df2-6027-47f8-87c5-3d6776ffca87%2Finbox-layout.gif&blockId=21623bf6-7c08-4523-89d4-6a175725ecbf&projectId=1032)</Frame>

The Inbox layout is configured similar to the other [List](/building-blocks/vikC2AWEpQGkZd4jGyoVxo/list-block/pQhPYuA28VXr5tLLFBcbCU) layouts, but the field mapping part has some nuances. So, let's take a closer look at that.

## **Mapping the fields**

The layout has a set of fixed fields (4 of them under **List Item**) that represent the values in the left sidebar.
<Frame caption="List item fields">![List item fields](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F63ac03cb-00bf-456c-8878-5532f82d91b9%2Flist-item-fields.png&blockId=27638df7-9834-4702-b8cf-7895f441b392&projectId=1032)</Frame>

Below, we've mapped those fields to an [Airtable](/data-sources/1YYStCJukT15Lh2j93qo9A/airtable/jHX7kfH7QvkVcELQvS3VoX) base, and you can see how they're displayed inside the sidebar. Note that one of the fields is hidden, so you can show/hide them based on your needs.
<Frame caption="Mapping the sidebar fields">![Mapping the sidebar fields](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F1d6dbe7c-4043-4028-992b-cd01291d4f33%2Ffields-mapped.png&blockId=14533992-4787-41f7-b0b3-436693ee2478&projectId=1032)</Frame>

Next, you have the **List Detail Fields** (you can add or remove as many fields as you wish) that represent the details section of the list item. Below, you can see that we've used two fields, and now their values are displayed in the details window for each list item. **The fields we've used are different from those that we used for the sidebar part, but you can use the same fields again.**
<Frame caption="Mapping List Item Fields">![Mapping List Item Fields](https://media.helpkit.co/proxy?url=https%3A%2F%2Fs3-us-west-2.amazonaws.com%2Fsecure.notion-static.com%2F4b68632e-7b41-41e3-bd64-4978da562700%2Fmapping-list-item-fields.png&blockId=0ef4c535-69ed-4bac-975f-9cedce74bffb&projectId=1032)</Frame>
