---
title: "Create a Database"
description: "Create a new database in the specified workspace."
api: POST https://tables-api.softr.io/api/v1/databases
contentType: application/json
---


<ResponseExample>
```json Request Body
{
  "workspaceId": "string",
  "name": "string",
  "description": "string"  // optional
}
```

```bash Example Request
curl -X POST 'https://tables-api.softr.io/api/v1/databases' \
  -H 'Softr-Api-Key: <TOKEN>' \
  -d '{
    "workspaceId": "<WORKSPACE_ID>",
    "name": "Marketing DB",
    "description": "Stores campaign data"
	}'
```

```json Response
{
  "data": {
    "id": "string",
    "name": "string",
    "description": "string",
    "workspaceId": "string",
    "createdAt": "2025-06-01T10:42:10.000Z",
    "updatedAt": "2025-06-01T10:42:10.000Z"
  }
}
```
</ResponseExample>
