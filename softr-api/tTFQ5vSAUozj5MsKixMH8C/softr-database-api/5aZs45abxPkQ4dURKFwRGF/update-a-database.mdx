---
title: "Update Database"
description: "Update the details of an existing database."
api: PUT https://tables-api.softr.io/api/v1/databases/{databaseId}
contentType: application/json
---

<ResponseExample>
```json Request Body
{
  "name": "string",       // optional
  "description": "string" // optional
}
```

```bash Example Request
curl -X PUT 'https://tables-api.softr.io/api/v1/databases/<DATABASE_ID>' \
  -H 'Softr-Api-Key: <TOKEN>' \
  -d '{
    "name": "Updated Name",
    "description": "Updated description"
  }'
```


```json Response
{
  "data": {
    "id": "string",
    "name": "string",
    "description": "string",
    "workspaceId": "string",
    "createdAt": "2025-06-01T10:42:10.000Z",
    "updatedAt": "2025-06-01T10:42:10.000Z"
  }
}
```
</ResponseExample>
