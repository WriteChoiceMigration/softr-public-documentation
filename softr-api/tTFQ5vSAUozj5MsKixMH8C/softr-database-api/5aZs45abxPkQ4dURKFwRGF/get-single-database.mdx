---
title: "Get Single Database"
description: "Retrieve details of a specific database by its ID."
api: GET https://tables-api.softr.io/api/v1/databases/{DATABASE_ID}
contentType: application/json
---

```bash
curl -X GET 'https://tables-api.softr.io/api/v1/databases/<DATABASE_ID>' \
  -H 'Softr-Api-Key: <TOKEN>'
```
